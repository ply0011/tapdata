##dataset frontend dev env


* for frontend setup, in the bash env of the host 
  First time setup libs 
  * ```cd datacollector-ui```
  * ```npm i```
  * ```npm install -g bower```
  * ```npm install -g grunt-cli```
  * ```bower install```
 
  How to start:
  * ```grunt watch``` 
  * ```docker-compose up streamset``` 
  * visit URL: http//localhost:18630
